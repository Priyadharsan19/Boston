Difference Between HTML and HTML5

HTML HTML5
Older version of the markup language. Latest version with new features and APIs.
Does not support audio/video tags directly. Supports built-in <audio> and <video> tags.
Limited support for semantic elements. Introduces semantic tags like <article>, <section>, <nav>.
No support for local storage. Supports local storage and session storage.
Older doctype declaration: <!DOCTYPE HTML PUBLIC...> Simple doctype: <!DOCTYPE html>
No built-in support for canvas and SVG. Supports <canvas> and SVG for drawing graphics.
